# 주석 스타일 가이드

## 1. 파일 헤더 주석

모든 Python 파일 최상단에 작성합니다.
작성자, 마지막 수정일자, 파일 역할과 목적을 기록합니다.

```python
# Author: 홍길동
# Last Modified: 2025/01/01 by 홍길동
#
# 역할: 이 파일이 무엇을 하는지 (한 줄)
# 목적: 왜 이 파일이 필요한지 (한 줄)
```

## 2. 함수·클래스 설명 주석

함수나 클래스의 동작, 입력, 출력에 대한 설명은 `"""` docstring 형태로 작성합니다.
다음 양식을 사용합니다:

- **함수명**: 함수의 이름
- **설명**: 함수의 역할 및 동작
- **입력**: 매개변수와 그 역할, 타입
- **출력**: 반환하는 값과 타입

```python
def add(num1: int, num2: int) -> int:
    """
    함수명: add
    설명: 두 숫자를 더합니다.
    입력:
     - num1 (int): 첫 번째 숫자
     - num2 (int): 두 번째 숫자
    출력:
     - (int): 두 숫자의 합
    """
    return num1 + num2
```

## 3. 인라인 주석

동작 의도를 단순히 파악하기 어려울 때 `#` 을 사용해 작성합니다.
주석은 코드의 오른쪽 또는 위쪽에 작성합니다.

```python
# 변수 초기화
total = 0

# 1부터 10까지의 합을 계산
for i in range(1, 11):
    total += i  # 각 숫자를 누적

# 최종 결과 리턴
return total
```
